Non-Negative Matrix Factorization-Convolutional Neural Network (NMF-CNN) For Sound Event Detection

このような統合の重要な考え方は、NMFを使用して、弱いラベルのデータに近似の強いラベルを提供することです。このような統合により、ベースラインシステムと比較して高いイベントベースのF1スコアを達成できました（評価データセット：30.39％vs 。23.7％、検証データセット：31％対25.8％）..検証結果を他の参加者と比較することにより、提案されたシステムは、今年のタスク4チャレンジで19チーム（ベースラインシステムを含む）で8位にランクされました。
